##题目

[X数](http://acm.zjut.edu.cn/ShowProblem.aspx?ShowID=1147)

##分析

n!求第一个非0位？

`n!=乘积(非5倍数的数)*5^(n/5)*(n/5)!`，可以把5^(n/5)去掉，计算乘积(非5倍数的数)，计算之后除以2^(n/5)的，然后算个位。

乘积(非5倍数的数)的个位数计算出来，然后把2^(n/5)个位计算出来，其实直接可以得到 乘积(非5倍数的数)/(2^(n/5))的个位，因为2的个数肯定大于5的个数，所以结果肯定为偶数且不为0，那么只有2、4、6、8

设i = 个位（乘积(非5倍数的数)），j = 个位(2^(n/5))，那么(i,j)组合之后的个位数可以用一个数组直接得到

```
a[2][2]=6;
a[2][4]=8;
a[2][6]=2;
a[2][8]=4;
a[4][2]=2;
a[4][4]=6;
a[4][6]=4;
a[4][8]=8;
a[6][2]=8;
a[6][4]=4;
a[6][6]=6;
a[6][8]=2;
a[8][2]=4;
a[8][4]=2;
a[8][6]=8;
a[8][8]=6;
```

而2^(n/5)的个位可以用数组b表示：

```
b[0] = 6;
b[1] = 2;
b[2] = 4;
b[3] = 8;
```

而计算乘积(非5倍数的数)的个位时，因为`2*3*4*6*7*8*9`的个位为6，所以只要算出n%10，如果`n<10`，可以直接计算，如果`n>=10`，那么前面的个位数肯定为6，然后计算n%10的个位数即可，然后就可以很容易得到个位（乘积(非5倍数的数)）;接下去就是递归解决solve(n/5)了。
